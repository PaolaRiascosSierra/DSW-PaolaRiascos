Introducción a CSS y Frameworks CSS

Un sitio WEB, no solo esta constituido pot la estructura del mismo dada por html, también rquiere de un diseño característico cuyo objetivo no es unicamente que tenga una apariencia agradable sino que también brinde al usuario una mejor experiencia al momento de navegar en el sitio.
Actualmente existen muchas herramientas que evitaran muchas lineas de código para el estilo de un proyecto, ya que éstas cuentan con una serie de diseños prestablecidos ya codificados para diferentes tipos de componentes, logrando también ahorrar tiempo en el proceso de diseño.
Por estas razones, el propósito de ese sitio es dar a conocer la funcionalidad y utilidad de las hojas de estilo en Cascada, CSS y de las herramientas CSS en linea disponibles que facilitan el proceso de diseño en una página WEB.
